Help

Getting started

Use File → New to start a blank document or upload an existing PDF to edit. Your work saves automatically a moment after you stop editing — look for the save status next to the Download button in the header.

Adding content

The toolbar below the header lets you add text, rectangles, ellipses, borders, and images. Click any object on the page to select it — once selected, you can drag to move it, drag a corner handle to resize or rotate it, and use the toolbar controls (font, size, color, bold/italic/strikethrough, links) to style text.

Working with pages

Use the page list on the left to switch between pages, or + Add page to add a new one. Right-click any page for options to rename, duplicate, or delete it.

Undo and redo

Press Ctrl+Z to undo and Ctrl+Y (or Ctrl+Shift+Z) to redo. These are also available under the Edit menu.

Sharing a document

Use File → Share to generate a link with either view-only or edit access. Anyone with the link can open it without needing an account — treat edit links the same way you would treat sharing a password, since anyone holding one can change the document.

Working offline

You can view a document you've already opened while offline, but editing requires a connection to save your changes. If you lose connectivity while editing, you'll see an offline indicator in the header, and your edits will sync once you're back online.

Downloading your PDF

Click Download PDF at any time to export your document as a real, standalone PDF file you can save anywhere on your device.
